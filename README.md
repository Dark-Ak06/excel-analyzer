# AI Excel Analytics API

Backend сервис для автоматизации анализа Excel-данных.

## 🚀 Возможности
- Загрузка Excel файлов
- Автоматический анализ
- Генерация инсайтов
- REST API

## 📡 Пример запроса

POST /analyze

## 📊 Пример ответа

{
  "metrics": {...},
  "insights": [...]
}

## 🛠 Стек
- FastAPI
- Pandas
- Python

## 💡 Идея
Проект автоматизирует ручной анализ Excel, который обычно выполняется аналитиками вручную.
