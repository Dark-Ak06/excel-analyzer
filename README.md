# 📊 Excel Analyzer API

Backend-сервис для автоматизации анализа Excel-файлов с использованием Python, FastAPI и data-driven подхода.  
Проект предназначен для замены ручного анализа табличных данных и подготовки к интеграции LLM.

---

## 🚀 Возможности

- 📥 Загрузка Excel-файлов (.xlsx, .xls)
- 📊 Автоматический расчет метрик:
  - среднее (mean)
  - медиана (median)
  - сумма (sum)
  - минимум / максимум
  - стандартное отклонение
- 🧠 Генерация текстовых инсайтов
- 🧹 Анализ качества данных:
  - пропущенные значения
  - типы данных
- 🤖 Подготовка данных для LLM (LLM-ready архитектура)
- 📡 REST API (FastAPI)
- 📘 Автоматическая документация (Swagger / ReDoc)
- 🧪 Unit-тесты

## 🧠 Идея проекта

В реальной практике аналитики часто вручную:
- открывают Excel
- считают показатели
- делают выводы

Этот сервис автоматизирует весь процесс:

👉 Excel → обработка → анализ → инсайты → API

Проект может быть использован как основа для:
- автоматической отчетности
- аналитических сервисов
- интеграции с AI/LLM
